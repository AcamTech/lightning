Execution Benchmark

Lightning executed 400-500K URLs/min on a machine with 16GB RAM and 8 CPUs, which is really impressive as compared to any other URL execution framework in the open source environment we tested.


Comparative Study

Comparative study was done on similar projects.

Async-http-client (previously known as Ning) did not suit our requirement due to the below limitations

  • Could not achieve the required speed.
  • Not horizontally scalable.
  • Not distributed.

Though not used for batch execution, lightning uses Ning to make POST and PUT calls.
