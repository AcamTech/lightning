Lightning

Blazing Fast URL Execution Engine

Motivation

We were looking for a system that can periodically hit an extremely large number of URLs with minimum latency. But there were no libraries or frameworks out there that could support our use case. This motivated us to build a system of our own, that could be easily integratable, handle millions of URLs requests, execute with reliability and provide the results back via the simple client interface.
